# Linux System Administration Scripts — RHCSA Level

A collection of shell scripts and configurations from hands-on
Linux administration practice covering RHCSA-level topics.

## Topics Covered
- User and group management with ACLs
- LVM storage configuration
- Firewall rules with firewalld
- SELinux policy management
- Cron job scheduling
- SSH configuration
- System monitoring with journalctl

## Scripts

### user-management.sh — Create user with ACL permissions
### lvm-setup.sh — Create and mount logical volumes
### firewall-rules.sh — Configure firewalld rules
### cron-backup.sh — Automated backup using cron
